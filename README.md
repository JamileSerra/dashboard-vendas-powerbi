# 📊 Dashboard de Vendas e Lucratividade - Power BI

Este projeto consiste na criação de um dashboard interativo no Power BI para análise de vendas, lucro e desempenho por país, produto e segmento.

---

## 🎯 Objetivo

Analisar os principais indicadores de desempenho de vendas, proporcionando uma visão clara sobre:

- Volume de vendas  
- Lucro por país  
- Desempenho por segmento  
- Evolução das vendas ao longo do tempo  

---

## 📌 Funcionalidades

- 📍 Mapa com vendas e unidades vendidas por país  
- 💰 Mapa com lucro (profit) por país  
- 🍰 Gráfico de pizza com distribuição de lucro por segmento  
- 📈 Gráficos de tendência de vendas  
- 📊 Comparação entre produtos e segmentos
